# egg-wugs-framework

egg framework for myself

## QuickStart

```bash
npm i --save egg-wugs-framework
```

then update your package.json

```json
{
    "egg": {
        "framework": "egg-wugs-framework"
    }
}
```

so your can use these plugins without `npm i --save ...`

    "dependencies": {
